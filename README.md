# gas-CombinationTestCaseGenerator     

Google スプレッドシートで、組み合わせテストケースの生成をする。コンテナバインドスクリプトです。     

-------------------------------------------------------
## 生成できる組み合わせテストケースについて     
以下のテストケースの生成が可能です。    

* 全網羅テスト    

* ワンワイズ     

* ペアワイズ      

全網羅と、ワンワイズ テストの生成は、コンテナバインドスクリプトのみで可能ですが、ペアワイズテストは Google Cloud Functions のデプロイが必要になります。
